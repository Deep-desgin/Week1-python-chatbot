# Week 1 Python Chatbot

## Description

This project is a simple Python chatbot created for the Week-1 Python assignment.

The chatbot:

* Asks the user to enter their name
* Greets the user with a welcome message
* Responds to basic inputs like **hi**, **hello**, **how are you**, and **bye**
* Shows a default response for unknown messages
* Stops only when the user types **exit**

## How to Run

Run the program using:

python chatbot.py
